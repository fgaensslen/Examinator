---
question: "As a developer, how can you identify a JavaScript action on GitHub?"
documentation: "https://docs.github.com/en/actions/creating-actions/creating-a-javascript-action"
---
- [ ] The action’s repository includes a js.yml file in the .github/workflows directory.
- [ ] The action’s repository name includes the keyword “JavaScript.”
- [x] The action.yml metadata file has the runs.using value set to node16.
- [ ] The action.yml metadata file references a package.json file.
